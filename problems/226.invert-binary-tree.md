## 题目描述

翻转一棵二叉树。

示例：

输入：
```
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```

输出：
```
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```

## 标签

- 树

## 思路

### 解法1：递归

代码实现如下：
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @return {TreeNode}
 */
var invertTree = function(root) {
    if (root == null) return null;
    const right = invertTree(root.right);
    const left = invertTree(root.left);
    root.left = right;
    root.right = left;
    return root;
};
```

复杂度分析：

- 时间复杂度是 O(N)。
- 空间复杂度是 O(N)。
