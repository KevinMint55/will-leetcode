## 题目描述

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？

示例 1:
```
输入: m = 3, n = 2
输出: 3
解释:
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -> 向右 -> 向下
2. 向右 -> 向下 -> 向右
3. 向下 -> 向右 -> 向右
```

示例 2:
```
输入: m = 7, n = 3
输出: 28
```

提示：
- 1 <= m, n <= 100
- 题目数据保证答案小于等于 2 * 10 ^ 9

## 标签

- 数组
- 动态规划

## 思路

### 解法1：动态规划

- 令dp[i][j]是到达i, j最多路径
- 动态方程：dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
- dp[0][i]与dp[i][0]都是在边界，只能为1

代码实现如下：
```javascript
/**
 * @param {number} m
 * @param {number} n
 * @return {number}
 */

var uniquePaths = function(m, n) {
    const dp = [];
    for (let i = 0; i < m; i ++) {
        dp[i] = new Array(n);
        dp[i][0] = 1;
    }
    for (let i = 0; i < n; i ++) {
        dp[0][i] = 1;
    }
    for (let i = 1; i < m; i ++) {
        for (let j = 1; j < n; j ++) {
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
        }
    }
    return dp[m - 1][n - 1];
};
```

复杂度分析：

- 时间复杂度是 O(m * n)。
- 空间复杂度是 O(m * n)。
